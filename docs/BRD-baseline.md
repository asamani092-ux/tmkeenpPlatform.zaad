# BRD Baseline — منصة تمكين (v1)

> مستخرج من جدول التسليمات + مقارنة بالكود الحالي — للاعتماد من جمعية الزاد

## المراحل الخمس

| # | المرحلة | شرط القبول | الحالة الحالية |
|---|---------|-------------|----------------|
| 1 | التحليل والتصاميم | اعتماد wireframes + BRD | MVP UI موجود؛ وثائق رسمية تنتظر اعتماد الجمعية |
| 2 | الهوية وبوابة المستفيد | تسجيل + رفع ملفات + شريط تقدم | تسجيل/دخول/ملف رقمي ✅؛ رفع ملفات حقيقي ✅ (Sprint 1) |
| 3 | الإرشاد وبوابة المرشد | إدارة مستفيدين + جدولة + توصيات | لوحة مرشد + جلسات + مهام ✅؛ بريد SMTP ✅ (Sprint 2) |
| 4 | التدريب والتوظيف | مرحلية صارمة + كتالوج + ترشيح | فرص + تقديم ✅؛ مراجعة تقديمات ✅ (Sprint 3) |
| 5 | الإدارة والأثر | تجريب + تقارير + تسليم كود | لوحة مدير ✅؛ تقارير أثر ✅ (Sprint 4) |

## فجوات مغلقة في v1.0

- PostgreSQL للإنتاج (بدل SQLite)
- رفع CV/شهادات PDF على السيرفر
- بريد SMTP Hostinger
- مراجعة التقديمات (قبول/رفض)
- تقارير أثر 3–6 أشهر
- CI/CD + دليل نشر VPS

## شروط تعاقدية

- **الملكية الفكرية:** كامل السورس كود والتصاميم لجمعية الزاد — راجع `NOTICE.md`
- **الضمان:** سنتان دعم فني بعد الإطلاق (أخطاء برمجية)

## أدوار النظام

- **ADMIN:** pipeline، فرص، مرشدون، مستفيدون، متابعة، تقديمات، تقارير، إعدادات
- **GUIDE:** مستفيدون في مرحلة GUIDANCE فقط — جلسات، مهام، توصيات
- **BENEFICIARY:** ملف رقمي، فرص مرحلية، مهام، جلسات
