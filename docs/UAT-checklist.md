# UAT Checklist — منصة تمكين

## المرحلة 2 — المستفيد

- [ ] تسجيل مستفيد جديد ببيانات كاملة
- [ ] رفع CV (PDF) وشهادات بنجاح
- [ ] تسجيل الدخول والوصول للوحة المستفيد
- [ ] شريط التقدم يعكس المرحلة الحالية
- [ ] تعديل الملف الشخصي

## المرحلة 3 — المرشد

- [ ] المرشد يرى مستفيديه في GUIDANCE فقط
- [ ] جدولة جلسة (Drawer) مع رابط/موقع
- [ ] استلام بريد للمرشد والمستفيد بعد الجدولة
- [ ] إضافة مهمة وظهورها للمستفيد
- [ ] توصيات مهنية ودورات تدريبية

## المرحلة 4 — التدريب والتوظيف

- [ ] المستفيد يرى فرص مرحلته فقط
- [ ] التقديم على فرصة
- [ ] المدير يراجع التقديم (قبول/رفض)
- [ ] لا انتقال مرحلة دون اعتماد Admin

## المرحلة 5 — الإدارة

- [ ] Pipeline + إدارة مستفيدين + إسناد
- [ ] تصدير Excel للجداول
- [ ] تقارير الأثر (3–6 أشهر) + تصدير
- [ ] إعدادات بريد المرسل
